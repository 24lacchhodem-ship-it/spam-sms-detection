# 📩 Spam SMS Detection using Machine Learning

**dataset**: https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset

## 📌 Project Overview
This project is a Machine Learning-based system that classifies SMS messages as **Spam** or **Ham (Not Spam)**. It uses Natural Language Processing (NLP) techniques to analyze text data and predict whether a message is spam.

---

## 🎯 Objective
- Detect spam messages automatically
- Reduce unwanted and harmful SMS
- Provide a simple GUI for real-time prediction

---

## 🧠 Technologies Used
- Python
- Scikit-learn
- Pandas
- NumPy
- Tkinter (for GUI)
- Natural Language Processing (NLP)

---

## ⚙️ Project Workflow
1. Data Collection  
2. Data Preprocessing  
3. Feature Extraction (TF-IDF)  
4. Model Training  
5. Model Evaluation  
6. Model Comparison  
7. Best Model Selection  
8. GUI Development  

---

## 🧹 Data Preprocessing
- Removed null values  
- Removed duplicate records  
- Converted text to lowercase  
- Removed stopwords and punctuation  

---

## 🤖 Machine Learning Models Used
- Naive Bayes  
- Logistic Regression  
- Support Vector Machine (SVM)  

---

## 📊 Model Evaluation
- Accuracy Score  
- Confusion Matrix  

---

## 🏆 Best Model
The model with the highest accuracy was selected and used for final predictions.

---

## 🖥️ GUI Application
A user-friendly interface is built using Tkinter where:
- User enters a message  
- System predicts whether it is Spam or Not Spam
